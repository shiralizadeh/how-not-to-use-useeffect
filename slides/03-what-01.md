# What is it?

- `useEffect(callback, deps)`
- A hook that runs a **callback** under conditions
  - On **every** render
  - React **mounts** the component once
  - Detect **changes** in dependencies
- It lets you perform side effects (not in the body)
  - Fetch data
  - DOM changes
  - Timers
  - Javascript stuff
