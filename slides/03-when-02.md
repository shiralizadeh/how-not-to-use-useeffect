# Question: Which one will cause a browser block?

::left::

## setTimeout

```ts
function loop() {
  setTimeout(() => {
    loop();
  });
}

loop();
```

::right::

## Promise

```ts
function promiseLoop() {
  Promise.resolve().then(() => {
    promiseLoop();
  });
}

promiseLoop();
```
